## How to read this list

- Most `/api/*` endpoints require a logged-in session. State-changing browser requests also send a CSRF header.
- `/api/auth/login`, `/api/auth/register`, `/api/auth/refresh`, `/api/health`, and public share reads can be called without an active app session.
- Operator-only APIs map to operator UI areas such as system credentials, System LLM, and marketplace moderation.
