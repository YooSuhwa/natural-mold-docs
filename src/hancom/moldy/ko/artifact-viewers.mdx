Moldy artifact viewer는 파일의 `kind`, extension, MIME type을 조합해 preview provider를 선택합니다. 이 문서는 대화 artifact card, 오른쪽 artifact panel, **Settings > Files**, 공개 공유 화면에서 어떤 파일을 어떻게 볼 수 있는지 정리합니다.

기본 사용법은 [파일과 아티팩트](/hancom/moldy/ko/files-artifacts)를 참고하세요. 이 페이지는 파일 형식별 판정과 운영 체크에 집중합니다.

아래 갤러리는 Moldy artifact preview provider를 실제로 렌더링해 캡처한 화면입니다. 형식별로 어떤 preview가 열리는지 확인할 때 이 이미지를 기준으로 보면 됩니다.

## 선택 기준

| 기준 | 사용 방식 |
| --- | --- |
| `kind` | `image`, `video`, `audio`, `pdf`, `markdown`, `html`, `code`, `document`, `data`, `cad`, `other` 같은 상위 분류 |
| extension | `pdf`, `docx`, `xlsx`, `pptx`, `hwp`, `hwpx`, `csv`, `json`, `md` 등 실제 파일명 기반 판단 |
| MIME type | `image/*`, `text/html`, `application/pdf`, `application/json` 같은 content type |
| status | `ready`일 때 preview/download 가능 |
| size limit | text preview byte limit, maximum artifact bytes, maximum files per run의 영향을 받음 |

## 미디어 뷰어

| 파일 | preview | 주요 행동 | 주의 |
| --- | --- | --- | --- |
| PNG/JPG/JPEG/WebP | 이미지 preview와 확대 | 확대, 다운로드, 공유 전 확인 | 이미지 안의 개인정보와 내부 URL 확인 |
| MP3/WAV 등 audio | 브라우저 audio player | 재생, 다운로드 | 브라우저가 codec을 지원해야 함 |
| MP4/WebM 등 video | 브라우저 video player | 재생, 다운로드 | 큰 파일은 preview보다 download가 안정적일 수 있음 |
| PDF | embed preview | page 확인, 다운로드 | 브라우저 PDF viewer 정책에 영향 |

## 문서 뷰어

| 파일 | preview | 주요 행동 | fallback |
| --- | --- | --- | --- |
| Markdown | Markdown renderer | heading, table, link 확인 | raw text |
| Mermaid | diagram renderer | flowchart/sequence 확인 | 코드 블록 |
| HTML | sandboxed iframe | 렌더링 확인 | 다운로드 |
| DOCX | document preview | 본문, 표, 제목 확인 | 다운로드 |
| XLS/XLSX | sheet/table preview | sheet, row, column 확인 | 다운로드 |
| PPTX | slide preview | slide 흐름 확인 | 다운로드 |
| HWP/HWPX | 한글 문서 preview | 본문 구조 확인 | 다운로드 |

<Warning>
HTML preview는 sandboxed iframe으로 열립니다. 그래도 외부 링크, embedded media, 민감한 본문이 포함될 수 있으므로 공유 전 내용을 확인하세요.
</Warning>

## 데이터와 코드 뷰어

| 파일 | preview | 확인할 것 |
| --- | --- | --- |
| CSV/TSV | table preview | delimiter, header, row count |
| JSON | 구조화 preview | key 이름, array/object 깊이, 민감 field |
| YAML/TOML | key/value tree | 설정값, secret placeholder 여부 |
| TXT/LOG | text preview | error line, stack trace, 내부 경로 |
| code artifact | code preview | language, generated path, overwrite 위험 |
| CAD/기타 | fallback card | download 가능 여부 |

## 대화, 라이브러리, 공유의 차이

| 위치 | 가장 잘 보는 것 |
| --- | --- |
| 채팅 artifact card | 방금 생성된 파일과 실행 맥락 |
| 오른쪽 artifact panel | 같은 대화의 파일 목록과 preview |
| **Settings > Files** | 여러 에이전트/대화의 파일 검색, 필터, favorite, stats |
| 공개 공유 화면 | share token으로 노출되는 read-only snapshot과 공유 파일 |

## 실패 판정

| 증상 | 우선 확인 |
| --- | --- |
| preview가 비어 있음 | artifact status, MIME type, extension, size limit |
| 다운로드가 안 됨 | storage object 존재, current version, owner permission |
| 공유 화면에서 안 열림 | active share token, artifact가 공유 대화에 속하는지 |
| 문서가 fallback만 보임 | 해당 extension/MIME provider 지원 여부 |
| 오래된 파일이 보임 | version number와 update time |

## 운영 체크리스트

파일 viewer 기능이 추가되거나 변경되면 다음을 같이 갱신하세요.

1. 이 매트릭스의 extension/MIME type 행.
2. [파일과 아티팩트](/hancom/moldy/ko/files-artifacts)의 요약 표.
3. [채팅 컴포넌트 갤러리](/hancom/moldy/ko/chat-components)의 artifact 카드 설명.
4. 공개 공유와 보안 주의.
5. `src/llms.txt`의 Files/artifacts 설명.
