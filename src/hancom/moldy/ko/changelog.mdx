---
title: 변경 이력
description: Moldy 문서 사이트의 주요 변경 사항을 기록합니다.
---

이 변경 이력은 소스 근거, 화면 캡처, API reference, 사용자 안내에 영향을 주는 문서 변경을 기록합니다. 항목에서 애플리케이션 변경이라고 명시하지 않는 한 제품 릴리스 노트가 아니라 문서 변경 이력입니다.

## 2026-06-08

- 채팅 컴포넌트 갤러리와 아티팩트 뷰어 매트릭스에 실제 Moldy 컴포넌트/preview provider 캡처를 추가하고 `capture:ui` 재생성 스크립트를 만들었습니다.
- 문서 변경 시 함께 갱신해야 하는 페이지와 산출물의 동시 갱신 규칙을 `AGENTS.md`와 릴리즈 문서 업데이트 가이드에 추가했습니다.
- 사용 시나리오와 레시피, 채팅 컴포넌트 갤러리, 아티팩트 뷰어 매트릭스, API Cookbook, 오류 메시지 사전, 보안과 프라이버시, 릴리즈와 문서 업데이트, FAQ 문서를 한국어/영어로 추가했습니다.
- 상단 우측에 Moldy GitHub 저장소 링크를 추가하고 `llms.txt`에도 source repository를 기록했습니다.
- 로컬 GEO/SEO 경량 점검을 실행하고 `llms.txt`, `robots.txt`, GEO 갱신 절차, production 배포 후 남은 sitemap/JSON-LD 작업을 문서화했습니다.
- 기능 메뉴의 Tools, MCP Servers, Skills가 각각 무엇을 관리하고 채팅에서 어떤 결과로 보이는지 설명을 보강했습니다.
- 에이전트 채팅 가이드에 Markdown, 코드 블록, Mermaid, KaTeX, 이미지, 검색 결과, 파일 도구, 승인, 사용자 입력, 메모리, 서브에이전트, Builder 카드 같은 인라인 컴포넌트 목록을 추가했습니다.
- 파일과 아티팩트 가이드에 이미지, 오디오/비디오, PDF, HTML, Markdown, Mermaid, 코드, 텍스트, CSV/TSV, JSON, YAML/TOML, HWP/HWPX, DOCX, XLS/XLSX, PPTX, fallback 다운로드 뷰어를 정리했습니다.

## 2026-06-07

- 현재 메뉴 구조에 맞춰 기본 사이드바, Settings 사이드바, 이동된 리소스 페이지, 운영자 전용 Admin 페이지를 문서에 반영했습니다.
- 앱 설정과 내비게이션, 메모리, 파일과 아티팩트, Agent API 가이드를 영어와 한국어로 추가했습니다.
- live OpenAPI의 205개 endpoint 표면에 맞춰 `agent-api`, `agent-runtime-api`, `artifacts`, `audit`, `memory` 생성 메타데이터를 보강했습니다.
- Settings, Memory, Files, Agent API, Activity Log, Appearance, Security, All Activity 화면 캡처 범위를 확장했습니다.

## 2026-06-02

- 영어와 한국어 문서에 E-E-A-T, AI 인용 가능성, answer-first 요약, 구조화 데이터 권장안을 보강하면서 기존 명령어, 링크, 스크린샷, API 동작 설명은 유지했습니다.
- 외부 제품 문서의 정보 구조를 참고하되 Moldy 소스에 있는 기능만 기준으로 문서를 확장했습니다.
- 빠른 시작, 주요 개념, 에이전트 설정, 도구, MCP 서버, 스킬, 마켓플레이스 설치/게시, 접근 권한과 감사, API 인증, 채팅 스트리밍 문서를 추가하거나 대폭 보강했습니다.
- MCP probe/import/export, 마켓플레이스 install/publish/admin listing, 공유 링크, 스케줄 trigger, usage aggregate, model fallback처럼 실제 라우트/API에서 확인된 기능을 상세화했습니다.
- Moldy 전용 문서 사이트를 한국어와 영어 구조로 구성했습니다.
- 현재 프론트엔드 라우트와 실행 중인 OpenAPI를 기준으로 기능 인벤토리와 API reference를 생성했습니다.
- Playwright로 로그인, 대시보드, 에이전트 생성, 채팅, 설정, 도구, MCP 서버, 스킬, 모델, 자격증명, 시스템 설정, 스케줄, 사용량, 마켓플레이스, 운영자 관리 화면을 캡처했습니다.
- 운영자 초기 설정, System LLM, 모델/자격증명, 문제 해결, 문서 갱신 워크플로를 문서화했습니다.
- 민감정보 검사, 이미지 참조 검사, broken-link 검사를 통과한 상태로 문서 서버를 확인했습니다.
