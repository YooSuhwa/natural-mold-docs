이 문서는 실행 중인 Moldy 백엔드의 `/openapi.json`에서 저장한 `src/openapi/moldy-openapi.json`를 기준으로 생성했습니다. 현재 스냅샷에는 205개의 endpoint가 있습니다.

<Info>
스키마, request body, response body의 상세 필드는 API Reference 탭의 OpenAPI 뷰에서 확인하세요. 이 페이지는 전체 API 표면을 빠르게 훑기 위한 색인입니다.
</Info>

## 읽는 법

- 대부분의 `/api/*` endpoint는 로그인 세션이 필요합니다. 상태를 변경하는 요청은 브라우저 앱 기준 CSRF 헤더도 함께 보냅니다.
- `/api/auth/login`, `/api/auth/register`, `/api/auth/refresh`, `/api/health`, 공개 공유 링크 조회 endpoint는 세션이 없을 때도 호출될 수 있습니다.
- 운영자 전용 기능은 시스템 자격증명, 시스템 LLM, 마켓플레이스 운영자 관리처럼 UI에서도 운영자 메뉴로 노출되는 영역입니다.
